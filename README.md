# Sales-Dashboard-Tableau

A dynamic, user-friendly European sales dashboard in Tableau for real-time insights, enhancing data visualization and decision-making capabilities.

### Data Description
The client will send data in multiple sets of files in batches at a given location.
Features:
<ul>
- Row ID: Unique identifier for each row in the dataset.<br>
- Order ID: Identifier for each sales order transaction.<br>
- Order Date: The date when the sales order was placed.<br>
- Dispatch Date: The date when the order was dispatched for delivery.<br>
- Delivery Mode: Mode used to deliver the products to customers.<br>
- Customer ID: Unique identifier for each customer in the dataset.<br>
- Customer Name: Name of the customer placing the sales order.<br>
- Segment: Categorization of customers based on certain characteristics or attributes.<br>
- City: Location where the customer is based or where sales occurred.<br>
- State/Province: Administrative region associated with the customer's location.<br>
- Country/Region: Country or region where the customer is located.<br>
- Region: Geographical region classification for sales analysis purposes.<br>
- Product ID: Identifier for each unique product in the dataset.<br>
- Category: Broad classification of products (e.g., electronics, furniture).<br>
- Sub-Category: More specific classification within a broader product category.<br>
- Product Name: Name or description of the specific product being sold.<br>
- Sales Quantity: Quantity of products sold in each transaction.<br>
- Discount: Reduction applied to the original product price during the sale.<br>
- Profit: Financial gain or loss associated with each sales transaction.<br>
</ul>
